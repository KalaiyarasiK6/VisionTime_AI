# 👁️ VisionTime_AI
KackZen 2026
Team - ProX

An AI-powered real-time surveillance system that detects suspicious activities using YOLO and ByteTrack while providing a timeline of important events.

## 📌 Problem Statement

Traditional CCTV systems require constant human monitoring and make it difficult to quickly locate important events. VisionTime_AI automates surveillance by detecting, tracking, and organizing suspicious activities into an event timeline.

---

## ✨ Features

- 🎯 Real-time object detection using YOLO
- 👥 Multi-object tracking using ByteTrack
- 📹 Live webcam and video support
- 📝 Automatic event timeline generation
- 📊 Detection confidence display
- 💻 User-friendly web interface
- 📂 Event history management

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/KalaiyarasiK6/VisionTime_AI.git
```

### Open Project

```bash
cd VisionTime_AI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run

```bash
python app.py
```
